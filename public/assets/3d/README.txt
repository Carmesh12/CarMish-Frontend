CarMesh — local 3D files

Put models here. They are served from the site root as /assets/3d/<filename>

Supported: .glb, .gltf, .obj, .stl

Preview in the app (dev):
  http://localhost:5173/local-3d
  http://localhost:5173/local-3d?src=/assets/3d/demo.glb

Default in code: /assets/3d/demo.glb — add that file or pass ?src=
